# React Native Expo: Aplicaciones Nativas para iOS y Android

Este repositorio contiene los proyectos y ejercicios realizados durante el curso **"React Native Expo: Aplicaciones nativas para IOS y Android"** impartido por Fernando Herrera en DevTalles. 

El objetivo principal de este curso es adquirir bases sólidas sobre el framework de Expo para construir aplicaciones móviles profesionales que funcionen de forma nativa tanto en iOS como en Android.

## 🚀 ¿Qué aprendí en este curso?

A lo largo del curso se cubrieron temas fundamentales y avanzados para el desarrollo móvil moderno, incluyendo:

*   **Fundamentos:** Reforzamiento de React y uso estricto de **TypeScript**.
*   **Diseño y Estilos:** Uso de Flexbox, estilos nativos y **NativeWind**.
*   **Navegación:** Implementación de Stack, Tabs, Drawer y el nuevo **Expo Router**.
*   **Gestión de Estado:** Uso de Context API y **Zustand**.
*   **Peticiones HTTP:** Manejo de datos con Axios y consumo de servicios mediante **TanStack Query** (Query y Mutation).
*   **Funciones Nativas:**
    *   Uso de **Mapas** y Geolocalización.
    *   Acceso a la **Cámara y Galería** de imágenes.
    *   Configuración y envío de **Push Notifications** (Client & Server).
*   **Seguridad:** Autenticación con **JWT**, protección de rutas y almacenamiento seguro.
*   **Despliegue:** Preparación para tiendas (App Store y Play Store) usando **EAS Build**.
